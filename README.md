# DEMO
## 业务目标：
1. 降低门槛：一步式启动强化学习训练，只需简单的交互即可训练领域垂类agent；
2. 提高效率：通过skill明确边界，确保训练的高效运行，自动解决报错问题；
3. 可复现性：规范训练流程，实现训练策略可复现。
4. 可观测性：可实时评估训练状态
5. 自优化：可自动优化训练前（如库依赖，配置冲突等）和训练中的问题（如OOM（识别OOM报错后，自动下调batchsize等），超参调整（checkpoint，修改参数后重启）---待验证）
## 目标链路：
1. 读取用户提供的空白目录路径
2. 与用户交互，确认 AGENT_TYPE
3. 联网搜索对应数据集并让用户确认
4. 在该路径配置基础环境
5. 下载数据集并转换为 train.parquet/test.parquet
6. 下载模型并启动 PPO 训练
7. 若报错，按“即时修复清单”处理后重试
